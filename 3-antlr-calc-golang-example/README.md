# Antlr4 calulate Example in golang

## Install antlr4

[Install antlr4](https://github.com/antlr/antlr4/blob/master/doc/getting-started.md)

### Install antlr4 golang runtime

```
go get github.com/antlr/antlr4/runtime/Go/antlr
```

## build example

```
make
./visitor_example
./listener_example
```
